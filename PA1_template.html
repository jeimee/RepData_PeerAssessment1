Imputing missing values

#Calculation of the number of missing values
s <- sum(is.na(df))
perc <- mean(is.na(df$steps)) * 100 
perc <- format(round(perc,2), nsmall=2)

There are 2304 amount of missing values in this data. These values are only found in the steps column. So, 13.11 % rows of the steps column has missing values.

#Replace missing values with the average steps per interval.
df_new <- df %>% 
    group_by(interval) %>%
    mutate(steps = replace(steps, is.na(steps), mean(steps, na.rm = TRUE)))
